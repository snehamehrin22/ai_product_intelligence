## automation_core

Reusable **config + auth + connectors** for automations that need to talk to:
- Google Drive / Google Sheets (OAuth or Service Account)
- Notion (API token)

This is meant to be installed into any project in this repo via editable install:

```bash
pip install -e ./automation_core
```

See `AUTOMATION_SETUP.md` at the repo root for laptop + server setup.
